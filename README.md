## Prerequisites for MaterialYou (A12)
1. Android 12 DP2 or above.
2. Gboard.  _beta for now_
3. Custom recovery.
4. Android 12 gxoverlay enabled.

## Prerequisites for Google Branding on Spacebar (<A12)
1. Custom recovery.
2. Gboard _beta for now_
